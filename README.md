# share

Suite of tools to support domain name related investigations. Developed
alongside [Jigsaw Domains](jigsawdomains.com). Deliberately framed as a set of
distinct interacting command line tools, to permit and encourage a range of
experiments and investigations. Interesting use cases, general commentary, or
simply defect reports, are all welcome.

## Dependencies

### Operating System

Linux, and to a lesser extent, Debian 12, is assumed throughout.

### Packages

Install package dependicies as follows:
```{.bash}
sudo apt-get update
sudo apt-get install python3
sudo apt-get install python3-venv
sudo apt-get install python3-dev
sudo apt-get install gcc
sudo apt-get install mariadb-server
sudo apt-get install mariadb-client
sudo apt-get install rdap
```

### Virtual Python

Install and configure Virtual Python as follows:
```{.bash}
sudo apt-get update
sudo apt-get install python3
sudo apt-get install python3-venv
sudo apt-get install python3-dev
sudo apt-get install gcc
sudo apt-get install mariadb-server
sudo apt-get install mariadb-client
sudo apt-get install rdap
```
